# Button Locations & Flex Cable Routing

## iPad mini (6th gen) — A15 Chip

**Physical Button Layout:**
```
     [Power]
        │
   ┌────┴────┐
   │         │
   │  Screen │
   │         │
   │         │
   └─────────┘
      [USB-C]
```

**Internal (simplified):**
- Power button flex cable runs from the top-right edge → under the display adhesive → connects to the logic board near the top-right corner.
- Volume buttons run along the opposite edge.

**Common failure points:**
1. Drop causing housing bend → button pressed against frame
2. Flex cable connector loosened (requires opening)
3. Button actuator (small metal dome) cracked

## iPad Air (5th gen) — Similar to mini 6

Slightly larger, but button layout and internal routing are conceptually identical.

## Repair Difficulty

| Component | Difficulty | Risk |
|---|---|---|
| Clean external button | Easy | Low |
| Realign bent housing | Medium | Medium |
| Replace power button flex | Hard | High (requires screen removal) |
| Replace logic board connector | Expert | Very High |

## Notes

- iPads are NOT designed for user-replaceable buttons.
- Apple Authorized Service Provider is the safest route.
- Third-party repair shops can do it cheaper but may use non-OEM parts.
